Gradle

Gradle requires Java JDK or JRE version 7 or above to be installed.

Download by wget the compatible distribution required by the project (V.3.x), preferably downloading the last stable distribution from the Gradle Distributions page. 

$ wget https://services.gradle.org/distributions/gradle-5.0-bin.zip -P /tmp
    or
$ wget https://services.gradle.org/distributions/gradle-3.5.1-bin.zip -P /tmp

Unzip in the environment folder into /opt/ inside a new (or existing) folder gradle/ 

$ sudo unzip -d /opt/gradle /tmp/gradle-*.zip
                    
$ ls /opt/gradle/gradle-
Edit profile file to set Gradle resource, replacing the XX for the proper version (It also works to alternate versions) 
$ sudo gedit /etc/profile.d/gradle.sh
Pasting the following code: 
# export GRADLE_HOME=/opt/gradle/gradle-6.XX
# DotHR Proyect compiler:
export GRADLE_HOME=/opt/gradle/gradle-5.XX
#export GRADLE_HOME=/opt/gradle/gradle-3.XX
export PATH=${GRADLE_HOME}/bin:${PATH}
Add executable permission to shell script and apply changes to account profile 
$ sudo chmod +x /etc/profile.d/gradle.sh
                    
$ source /etc/profile.d/gradle.sh

Verify the Gradle installation #

To validate that Gradle is installed properly use the gradle -v command which will display the Gradle version: 

gradle -v

You should see something like the following:

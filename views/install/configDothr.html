Hadoop 

Se sube el modelo Dummy en la carpeta creada en el EC2 Modeler.
                    Se sube el modelo en hadoop de hadoop. Recordando que el servidor de Hadoop debe estar arriba.
                    
                        $ hadoop dfs -mkdir /user/dothr/dothr-data
                        $ hadoop dfs -put /home/dothr/svn/dothr_otros/modelos/modelo_dummy/coreModel  /user/dothr/dothr-data/

Solr 

1) Copy svn://66.135.55.150/svn/dothr_otros/Solr/Solr_4.6.1/solr.war into /opt/solr/webapps
                    Ensure that these libs are included within this war file, if not, add them.
                    
                        commons-configuration-1.6.jar
                        commons-math3-3.2.jar
                        hadoop-core-1.2.1.jar
                        mahout-core-0.9.jar
                        mahout.math-0.9.jar
                    
                    Note : You can find these libs on 
                        /user/lib/mahout
                        /user/lib/mahout/lib
                        /user/lib/mahout/hadoop
                    
                    Add folder "classes" and its contents (log4j.properties) into war file solr.war into "WEB-INF" directory
                    
                        $ sudo cp /home/dothr/svn/dothr_otros/Solr/Solr_4.6.1/solr.war /opt/solr/webapps
                    
                    2) Copy svn://66.135.55.150/dothr_otros/Solr/Solr_4.6.1/core into /opt/solr/solr
                        $ sudo cp -r /home/dothr/svn/dothr_otros/Solr/Solr_4.6.1/core /opt/solr/solr/
                    
                    3) Copy svn://66.135.55.150/dothr_otros/Solr/Solr_4.6.1/coreModel into /opt/solr/solr
                        $ sudo cp -r /home/dothr/svn/dothr_otros/Solr/Solr_4.6.1/coreModel /opt/solr/solr/
                    
                    4) delete /opt/solr/solr/collection1
                        $ sudo rm -rdf /opt/solr/solr/collection1

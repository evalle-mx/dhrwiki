Install and manage multiple Java JDK and JRE versions on Ubuntu 20.04 

#Instalacion por PPA

                    1.Add the JAVA JDK PPA using the add-apt-repository comamnd.
                       $ sudo add-apt-repository -y ppa:webupd8team/java
                       $ sudo apt-get update
                   
                    2. Now, install Java JDK using the apt-get command.
                    During the installation, you would need to accept the Oracle binary licenses.
                       $ sudo apt install -y oracle-java8-installer
                   
                   contestar ok y despues yes
                   
                    3. revisar si se instalo
                      $ java -version
                   
                       dothr@dothr-asus:~$ java -version
                       java version "1.8.0_201"
                       Java(TM) SE Runtime Environment (build 1.8.0_201-b09)
                       Java HotSpot(TM) 64-Bit Server VM (build 25.201-b09, mixed mode)

To switch between different JDK Environments: 

$ sudo update-alternatives --config java
$ sudo update-alternatives --config javac
Both java and javac must point the same distribution 
$ java -version
$ javac -version

Also change and apply to environment $JAVA_HOME 

$ sudo gedit /etc/environment
$ source /etc/environment
$ echo $JAVA_HOME
